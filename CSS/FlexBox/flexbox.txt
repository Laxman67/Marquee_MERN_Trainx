What is Flexbox:
Flexbox Box model is a one dimensional (1-D) layout model that provides a method to align and arrange element in a very effective ways ( ROWS and COlUMNS) - One at a time

===== Flexbox Container and Flex items =====
Flexbox Container: Its is a element that holds flex items
Flex Items :  Items are the Direct children of container

flex container is created using
[ display:flex ]

Some important Properties"

=======FLEX CONTAINER PROPERTIES=======

flex-direction* : row,column,row-reverse,column-reverse
flex-wrap*:  to wrap the child element in screen width
justify-content* : main-axis (end,start,center,space-around,space-between) -> default is start
align-items* : cross-axis (end,start,center) - > default is strech



=FLEX ITEMS PROPERTIES=
order*
align-self*
--------
flex-grow
flex-shrink
flex-basis ( same as width of the box )
------------
  /* flex grow,shrink, basis */
  flex:1 1 100px;
  flex :1

  flex-grow: Defines the ability for a flex item to grow if necessary.
flex-shrink: Defines the ability for a flex item to shrink if necessary.
flex-basis: Defines the default size of an element before the remaining space is distributed.
